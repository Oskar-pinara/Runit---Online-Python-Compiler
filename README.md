# RunIt — Python Online Compiler

> Code it.Run it.Nail it.
> Fast, minimal Python compiler that runs entirely in your browser. No setup. No account. Just code.

## ⚡ Features
- Instant Python execution with zero configuration required
- Syntax highlighting, autocomplete, and error detection
- Dark & Light mode support
- Fully responsive across all devices

## 🚀 Getting Started
Open the website and start writing Python code immediately. No account or installation required.
